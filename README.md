# Desafio-de-Projeto-GiTHUB-primeiro-repositório
Primeiro projeto entregue durante o curso git/github DIO

## A idéia deste projeto é que o aluno do curso git/github da DIO, crie seu primeiro repositório no Github, faça o clone do repositório em sua máquina local, crie conteúdos, pastas, e envie essas atualizações novamente ao repositório no Github. O objetivo é exercitar os comandos que foram apresentados em aula.
